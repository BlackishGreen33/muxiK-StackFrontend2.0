## 🔖 项目导览

### ✨ 提交规范

- 🎉 init：项目初始化
- ✨ feat：新增功能（feature）
- 🐞 fix：修复bug
- 📃 docs：文档修改
- 🌈 style：代码样式修改，不影响原代码逻辑
- ✅ test：测试相关的改动
- 🔨 refactor：代码重构
- 🔧 chore：建制过程或辅助工具的变动

### 🎯 相容环境

- 现代浏览器（Chrome >= 64, Edge >= 79, Firefox >= 78, Safari >= 12）

### 💻 本地调试

```bash
$ git https://github.com/XiaoYouYunDong/XiaoYouYunDong-FE.git
$ cd ZhiHuiYuYan-FE
$ yarn
$ yarn dev:weapp
```
